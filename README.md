# sportsz
SportsZ is a web-based sports performance and talent management platform. It allows athletes to track, measure, and improve their performance through features like running speed tracking (manual or camera-based), goal-oriented dream boards, and motivational quotes. Users can register, upload performance videos, and showcase their achievements in community feeds. Coaches and evaluators can filter athletes, analyze performance history, and generate reports. Additional features include eligibility guidance, AI coaching tips, and gamified progress tracking with badges and confetti celebrations.

Key Focus: Athlete growth, performance analytics, and community engagement.
